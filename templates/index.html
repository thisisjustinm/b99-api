How to use

  1. /name/:name - Use name from the above list to get a random quote from any season. e.g /name/holt
  2. /name/:name?season=:season - Similar as above but with given season. e.g /name/holt?season=1
  3. /season/:season - Get a random quote from any character for the given season. e.g /season/1
  4. /season/:season?name=:name - Similar as above but with given name. e.g /season/1?name=holt
  5. /title/ - Returns a random TITLE OF A SEX TAPE. Link: /title
2 and 4 are essentially the same.
