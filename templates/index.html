How to use

S.No.EndpointInfoTry
1 /name/:name Use name from the above list to get a random quote from any season. /name/holt
2 /name/:name?season=:season Similar as 1 but with given season. /name/diaz?season=1
3 /season/:season Get a random quote from any character for the given season. /season/1
4 /season/:season?name=:name Similar as 3 but with given name. /season/1?name=peralta
5 /title/ Returns a random TITLE OF A TAPE /title
6 /title?name=:name Same as above but appends Title of :name's tape. /title?name=santiago
7 /insults/:name Returns a random insult by Holt/Wuntch but the name replaced with a given name. /insults/boyle
8 /alias Returns all the known aliases of Jake Peralta in a JSON format. /alias
9 /alias?season=:season Same as 8 but for a given season. /alias?season=5
