How to use

  1. /name/:name - Use name from the above list to get a random quote from any season. Try: /name/holt
  2. /name/:name?season=:season - Similar as above but with given season. Try: /name/diaz?season=1
  3. /season/:season - Get a random quote from any character for the given season. Try: /season/1
  4. /season/:season?name=:name - Similar as above but with given name. Try: /season/1?name=peralta
  5. /title/ - Returns a random TITLE OF A SEX TAPE. Try: /title
  6. /title?name=:name - Returns a random TITLE OF A SEX TAPE. But with given name. Try: /title?name=santiago
  7. /insults/:name - Returns a random insult by Holt/Wuntch, but the name replaced with a given name. Try: /insults/boyle
